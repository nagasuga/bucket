bucket
======

CLI for auto "bucket"ing files by moving to proper directory filtered by type of file.


Install
=======

```
pip install -U git+https://github.com/nagasuga/bucket.git
```


Usage
=====

Go to a git project and execute `bucket` in terminal

```
$ bucket filenames ...
```
